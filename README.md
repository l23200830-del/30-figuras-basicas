# 🎨 Geometric Studio | Canvas 2D Replication

Este proyecto es una exploración visual y técnica sobre la síntesis de gráficos mediante código. Utiliza el **Canvas 2D API** de JavaScript para recrear una composición urbana geométrica, empaquetada en una interfaz inspirada en los estándares de diseño de **Apple**.



## 🚀 Características del Proyecto
- **Arquitectura de Software:** Código JavaScript modularizado por componentes geométricos.
- **UI/UX Moderno:** Implementación de *Glassmorphism* y diseño responsivo con **Bootstrap 5**.
- **Tipografía Dinámica:** Animaciones de estiramiento de fuentes mediante **GSAP** y **ScrollTrigger**.
- **Performance:** Renderizado optimizado sin dependencias de imágenes externas para el dibujo.
